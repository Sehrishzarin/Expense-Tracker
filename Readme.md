# Expense Tracker Mini-App

A simple, persistent full-stack Web Expense Tracker mini-app featuring a responsive React frontend dashboard and a Node.js Express backend powered by a persistent SQLite database.

## Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine (v18 or higher recommended).

## Project Setup & Installation

Follow these steps to set up and run the application on a fresh machine.

### Step 1: Install and run

# Server 
- Open server folder in the terminal and run `npm i`
- Next runt he command `nodemon index.js`

# Client
- Open the folder client/client and run `npm i`
- Run the command `npm run dev`
