### How to run:
- open server folder in the terminal and run the command `npm i` and `nodemon index.js`
- open client/client folder in the terminal and run the command `npm i` and another command `npm run dev`

### Stack choice: 
- Since it was a small and light project, and i needed to make it quick i focused more on the creation efficiency. its pretty easy and fast to set up node js and react js,
- My second reason why i choose react js is that its a one-page application framework so i dont need to worry about reloads
- the worse choice to make this mini project for me would be trying to do all that with raw Javascript and html.
